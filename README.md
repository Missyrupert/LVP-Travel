# LVP Travel

Clean, responsive website for LVP Travel, a Scottish private travel and transfer company.

## What is included

- Private airport transfers, special occasions, golf outings, day trips, hotel pickups, and custom travel sections
- Popular destinations and pricing sections
- Mobile-friendly booking enquiry form
- Required field validation and success message
- Date picker limited to six months ahead

## Editing

The site is a simple static build:

- `index.html` contains the page content
- `styles.css` contains the visual design
- `script.js` contains form behaviour and date limits

## Deployment

No build step is required. Publish the project root as the site directory.
